## Como rodar a aplicação


1. Clone o repositório
```sh
git clone https://github.com/Ricardofreitasdev/ReactJs-Login.git
```


2. Entre na pasta do projeto
```sh
cd ReactJs-Login
```

3. Instale as dependencias 
```sh
yarn install
```

4. Execute a aplicação 
```sh
yarn start
```


## Recursos

### Front-end
